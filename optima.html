Optima
The Route Optimizer

Optima finds the optimal route a user should drive, based on their starting location, ending location, and up to four locations in between. This application is ideal for users who have a starting point and an ending point in mind, but are flexible in the order in which they travel to other destinations in between. Optima uses the Google Maps API to recommend the best overall route that a user should take after taking into account many factors such as travel time, distance, and the number of turns a user would need to make. Optima also provides directions, estimated travel time, and distance traveled (in miles) for each segment of the recommended route, as well as for the total route.

Optima was created by Carla Miles, Steven Willis, Courtnay Mcewen, and Masajji Patrick.
